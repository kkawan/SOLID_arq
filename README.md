# KawanShigueoWatanabe_AtiviadePatterns

Atividade individual de Design Patterns feita em TypeScript.

## Patterns escolhidos

Foram escolhidos os padrões mais simples entre cada categoria solicitada:

- Criacional: Prototype
- Estrutural: Adapter
- Comportamental: Strategy

## Como executar

Instale as dependências:

```bash
npm install
```

Execute diretamente com TypeScript:

```bash
npm start
```

Ou compile e execute a versão JavaScript:

```bash
npm run build
npm run start:dist
```
